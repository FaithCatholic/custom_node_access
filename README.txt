---SUMMARY---

Custom Node Access allows admins and authors to limit view access to particular nodes to specific user roles using the content edit form.

---REQUIREMENTS---

*None. (Other than Field UI)

---USAGE---

1. Install/enable the module in the /admin/modules section
2. "Manage fields" on a node/content type of your choice
3. Add ONE "Access" field with a machine name of "field_access"
4. Save field settings using the defaults present
5. Use the add/edit content form to limit a piece of content to be viewed only by the selected roles
